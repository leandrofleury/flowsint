# 07 - Analyst Notes

## Notas rápidas

- Ferramenta promissora para ASM e OSINT visual.
- O maior valor está na correlação entre entidades, não na descoberta bruta isolada.
- Deve complementar ferramentas CLI, não substituí-las.
- Bom candidato para laboratório e demonstração técnica controlada.
- Exige atenção especial a Docker, secrets, exposição web e retenção de dados.

## Pontos fortes

- Interface visual baseada em grafo.
- Boa aderência para investigação por pivôs.
- Potencial de integração com ferramentas OSINT conhecidas.
- Pode ajudar a explicar relações técnicas para times não especialistas.

## Pontos fracos

- Pode depender de várias integrações externas para entregar valor máximo.
- Requer disciplina operacional para evitar mistura de dados entre clientes.
- Não entrega, sozinho, relatório executivo pronto.
- Não substitui scanner de vulnerabilidade.

## Decisão atual

**Manter e avaliar.**

## Próxima ação recomendada

Subir em laboratório isolado, executar varredura de dependências/imagens e validar um fluxo simples com domínio próprio ou ambiente autorizado.
